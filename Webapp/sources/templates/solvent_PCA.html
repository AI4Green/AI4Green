About the Solvent Surfer

The Solvent Surfer is a selection tool that aims to promote the use of greener alternatives to common laboratory solvents. Selecting a Substitution Target from the dropdown menu will highlight the selected solvent on the map and suggest the closest solvents as alternatives. Points can also be selected by clicking. Each solvent is colour-coded according to its CHEM21 SHE ranking, but the colour can be changed using the Change Colour dropdown. Reaction specific solvent maps can be viewed using the Reaction Class dropdown, and a brief explanation of each class is available under the Reaction Class tab. Information about how the maps were generated is available below.


Principal Component Analysis

The Solvent Surfer is based on Principal Component Analysis (PCA) which is a statistical technique that reduces the dimensionality of large datasets. This is done by transforming input variables into Principal Components (PC's) that describe the variance of the data, often with the first two PC's describing the majority of this variance. This allows the data to be described in fewer dimensions without losing important information. These maps are generated using a variation of this technique called kernel PCA, which performs PCA analysis in a non-linear multidimensional space. The maps shown here were generated using a Gaussian kernel.

Solvent Dataset

The solvent maps are plots of PC2 vs PC1, which are generated through a PCA of a solvent descriptor dataset. This dataset describes 57 common laboratory solvents in terms of 16 descriptors that capture their physical and chemical properties. Of these solvents, 40 were included in the CHEM21 Solvent Selection Guide and are assigned their respective CHEM21 sustainability ranking. The remaining 17 solvents are unranked, but were included to show yield data for each reaction class. Reaction specific maps are generated by excluding redundant descriptors from the original dataset. Details on how this was done is given in the Reaction Class tab.
