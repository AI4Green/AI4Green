About AI4Green

AI4Green is open-source software developed in the group of Jonathan Hirst at University of Nottingham, UK. This work is supported by a grant from the Royal Academy of Engineering.



Visit our GitHub Repository for full source code and instructions on how to build your own instance of AI4Green.

Documentation & Help

For any queries, feedback, or suggestions, please contact us: ai4green@nottingham.ac.uk.
