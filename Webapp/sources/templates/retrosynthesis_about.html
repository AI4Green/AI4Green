About Retrosynthesis

❰ Return to Retrosynthesis

The retrosynthesis service has two core components: the search configuration and the reaction conditions.

Getting Started

To begin a retrosynthesis, you must enter a valid SMILES string for your target molecule into the 'SMILES' box. For a standard search of 100 iterations in 60 seconds, click 'New Retrosynthesis'. Your results will follow shortly after. You can view the suggested synthetic routes from the 'Route' dropdown. The results can be saved to your current workbook by using the 'Save to Workbook' option.


Configuration Options

The user has several options to configure the search towards their specific needs:

  1. Number of iterations: More iterations of the algorithm provides a more comprehensive search, but requires more time.
  2. Search time: One can specify the maximum run-time (in seconds) to suit one's requirements.
  3. Max depth: The max depth allows the user to control the maximum number of reaction steps for the suggested routes.
  4. Enhancement type: Different enhancements to the core algorithm can change the performance. One can select from 4 enhancement options:
    • Default: The default retrosynthesis method.
    • Enhanced Search: Uses the eUCT enhancement to adapt the style of search based-on how close the search is to termination.
    • Enhanced Speed: Uses the dUCT-v1 enhancement to focus on performing faster iterations, allowing more iterations in a given time limit.
    • Enhanced Solve Rate: Uses the dUCT-v2 enhancement to focus on solving more complex molecules.

Reactions and Conditions

Once your retrosynthesis is complete, you can view the produced routes in the 'Route' dropdown. On the left hand side you will find some information regarding the reaction conditions. One must select a molecule from the interactive display to view its conditions. The 'Compounds' tab will provide information on the selected molecule. The 'Reactions' tab will provide the conditions for the reactions to the selected molecule. You will receive predicted conditions for the following metrics:

  • Temperature
  • Solvent
  • Reagents
  • Stoichiometry/Catalyst
  • Element Sustainability
  • Atom Economy
  • Safety

The route tab displays an overview of the sustainability of each reaction step for each metric. This is also where the sliders are to adjust the weightings of the metrics which updates the weighted median used to colour reactions and routes. When calculating the sustainability metrics, we assume 1.0 equivalent of reagents and 0.05 equivalents of catalyst are used.
